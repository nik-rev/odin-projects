# Knight Travails

## What I Learned

- Using generator functions in javascript.

## Running locally

Run `node knight-travails.js`

## Features

- Finds a shortest path for a knight to travel between two cells on a chess board.

## Preview

```
> node knight-travails.js
( 8,8 ) --> ( 7,6 ) --> ( 8,4 ) --> ( 7,2 ) --> ( 5,3 ) --> ( 3,2 ) --> ( 1,1 )
```
